<h1>Beter Color</h1>

A Simple Project For BetterColor Readability. You Can Easily Choose The Best Color For Readability On Any Background.

# Technologies and Tools Used :
* React.Js
* Bootstrap
* CSS
* ColorPicker

# Key Features:
* Can Choose Colors From Color Picker
* Can Differentiate Color Of Text And Background According To The User's Wish
* Can Generate Color Codes (HEX CODES)
* Can Copy Color Codes
* Can Find Best Backgrounds For Suitable Text Color
